# Baekjoon_Omakase

2022 07 17 ~

백준 아이디를 입력하면 아이디의 수준에 맞는 3문제를 추천한다.  

#### 팀원
- 세종대학교 컴퓨터공학과 20학번 박규태(frontend(web))
- 세종대학교 전자정보통신공학과 18학번 박상욱(backend)

#### 개발언어  
##### frontend
react

##### backend
Node.js, Express

#### 개발기능
- [ ] 사용자의 백준 아이디 입력받기
- [X] 사용자가 푼문제 크롤링
- [X] 사용자 티어 크롤링
- [ ] 사용자가 푼문제가 아닌 다른 문제 추천
